Contrarian Investing Monitor

This page is only the dashboard/instructions page.
The Python stock screen does not run inside this web page. It runs in GitHub Actions on a schedule or when manually triggered.

Run the monitor

Open the GitHub Actions workflow and press Run workflow.

Open GitHub Action View Reports Folder View Python Script

What happens when it runs

StepAction
1Reads config/watchlist_asx.csv.
2Downloads recent price and market-cap data using yfinance.
3Flags stocks over A$500m market cap with sharp 1-day, 5-day, or 20-day falls.
4Writes a CSV report to reports/.
5Commits the report back into the repository.

Thresholds

TestDefault
Minimum market capitalisationA$500,000,000
1-day fall-7% or worse
5-day fall-12% or worse
20-day fall-20% or worse
Research aide only. This tool screens for potential contrarian opportunities. It does not recommend trades, verify solvency, check live announcements, or assess whether a fall is justified.
